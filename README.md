## Endpoints

### /random

GET `/api/v1/random`

```sh
curl https://pham-api.alexanderliu.dev/api/v1/random
```

**Response:**

```json
{
  "src": "https://pham-api.alexanderliu.dev/photos/<uuid>"
}
```
